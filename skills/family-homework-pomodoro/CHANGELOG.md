# family-homework-pomodoro changelog

This changelog tracks future updates for the `family-homework-pomodoro` skill.

## [1.0.2] - 2026-04-09

### Highlights

- clarify that the default `25/5` Pomodoro rhythm can be overridden by clear parent instructions
- add a sharper fit vs non-fit boundary for the public template
- expand the examples with larger-homework and unfinished-homework branches
- keep the public skill generic instead of turning it into a complex family scheduler

### Notes

- This is still a public-safe template, not a private household operating system.
- Multi-child parallel planning remains out of scope for the public version.

## [1.0.1] - 2026-03-26

### Highlights

- add a short example conversation
- add optional multi-child guidance
- refine parent-facing reward wording
- add a compact usage snippet for ClawHub and local OpenClaw installs

### Notes

- Keep the public skill generic and public-safe.
- Avoid adding household-specific or identity-specific details.
- Prefer documentation updates over behavior changes unless a workflow issue appears.
