# Publishing Notes

This skill is intended for the `media` execution flow.

`media` is the local instance / directory name.

`小果` is the Feishu bot name bound to that same executor.

## Preconditions

- `media` has permission to write to the target Feishu knowledge base
- the target `wiki_space` or `wiki_node` is explicit
- source content is public-safe and already reviewed

## Recommended process

1. Read the publish manifest.
2. Confirm the account is `media`.
3. Confirm the target wiki location.
4. Create or update pages in the defined order.
5. Save returned URLs for later reuse.

## Suggested page set

- 项目总览
- 对外发布文案
- 申请与介绍素材
