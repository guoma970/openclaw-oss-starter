# Feishu Knowledge Base Publisher

Compact skill for turning approved project pages into Feishu knowledge-base
documents.

## Use this skill when

- the target knowledge base is known
- the account has write access
- the source content is already approved

## Inputs

- one execution account such as `xiaoguo`
- one `wiki_space` or `wiki_node`
- one manifest describing titles, order, and source files

## Output

- created or updated Feishu pages
- returned document URLs
- brief publish summary
