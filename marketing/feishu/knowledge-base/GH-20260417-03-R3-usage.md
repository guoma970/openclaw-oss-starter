# GH-20260417-03-R3 使用说明

## 目标

基于固定 payload 预构建飞书批量写入计划，不直接写飞书。

唯一输入：

- `marketing/feishu/knowledge-base/payload-GH-20260417-02-R2.yaml`

## 脚本

- `marketing/feishu/knowledge-base/run_payload_batch_sync.sh`

## 默认行为

- 默认 `dry-run`
- 读取 payload 中的固定页面范围
- 输出结构化执行计划 JSON
- 如果切到非 `dry-run` 且 payload 中仍有权限 blocker，则直接阻断

## 用法

```bash
bash marketing/feishu/knowledge-base/run_payload_batch_sync.sh
```

指定 payload：

```bash
bash marketing/feishu/knowledge-base/run_payload_batch_sync.sh \
  marketing/feishu/knowledge-base/payload-GH-20260417-02-R2.yaml
```

模拟进入执行前检查：

```bash
BATCH_SYNC_MODE=ready \
  bash marketing/feishu/knowledge-base/run_payload_batch_sync.sh \
  marketing/feishu/knowledge-base/payload-GH-20260417-02-R2.yaml
```

## 预期输出

- `status = dry-run`，表示仅生成计划
- `status = blocked`，表示权限 blocker 尚未解除，禁止进入真实执行
- `plan.operations[]` 中逐条列出 update/create 分流结果

## 当前约束

- 不扩展 payload 之外的页面
- 不直接调用飞书写接口
- 权限 blocker 未解除前，不允许报真实执行完成
