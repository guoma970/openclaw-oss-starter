# 发布记录与复盘模板

## 使用方式

每次对外发布后，建议新增一条记录。

适用场景：

- GitHub 更新同步
- ClawHub 发布同步
- Feishu 群内同步
- 小红书等国内平台发布
- 其他外部渠道测试

## 模板

### 基本信息

- 发布日期：
- 发布主题：
- 对应项目页：
- 对应版本：

### 发布动作

- 发布平台：
- 使用文案：
- 是否做平台改写：
- 实际执行人：

### 反馈情况

- 是否有互动：
- 是否有收藏 / 点赞 / 访问增长：
- 是否带来后续问题或线索：

### 简短复盘

- 这次有效的点：
- 这次不够好的点：
- 下次保留什么：
- 下次调整什么：

## 轻量版模板

- 日期：
- 平台：
- 主题：
- 结果：
