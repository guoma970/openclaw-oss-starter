## 2. 对外一句话定位
A public-safe OpenClaw skill for recommending small, practical next-step growth actions for a public GitHub repository.

## 3. 核心卖点
- 不做空泛营销策划，专注下一步可执行动作
- 把仓库已有资产转成 1～7 天内能做的小动作
- 适合 release 后的 GitHub 仓维护、跟进、可见信号补强
- 适合技能型仓库和公共模板仓做后续增长收口

## 6. 可直接复用短版
新整理了一条公开 skill：`repo-growth-operator`。它不是让你做一整套营销计划，而是帮你从公开仓现状出发，找出接下来 1～7 天最值得做的几个小动作，比如补首页、补示例、发一次 release follow-up、准备社区回复。
